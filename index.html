How It Works

From zero to deployed app
in one session

Every workshop follows the same three steps. No experience needed — just show up and follow along.

01

Plan with Gemini AI

Tell Gemini what you want to build in plain English. It gives you the database design, API routes, and component structure before you write a line of code.

0 – 10 minutes
# Prompt Gemini:
"Design a to-do app with
 tags, due dates, and auth"
02

Build with GitHub Copilot

Open VS Code and start typing. Copilot suggests whole functions as you go. Ask it questions in plain English — it explains what the code does and why.

10 – 35 minutes
// Copilot writes this for you
async function getTodos() {
  return db.query('SELECT *')
}
03

Deploy with Docker Compose

One command starts your entire app — frontend, backend, and database together. No manual setup, no "works on my machine." It just runs.

35 – 50 minutes
$ docker compose up --build
✓ api   running on :3001
✓ web   running on :3000
✓ db    postgres healthy

The Toolkit

Three tools. One workflow.

Each tool does a specific job. Learn them together and you'll ship code the way professionals do.

Google Gemini

Your planning partner. Describe what you want to build and Gemini gives you a detailed blueprint — no experience needed.

  • Free via Google AI Studio
  • 1 million token context
  • Works with images too
  • API key in under 2 minutes

GitHub Copilot

Writes code as you type in VS Code. Suggests whole functions, explains errors, and answers questions — your always-on coding companion.

  • Free via GitHub Education
  • Works in VS Code
  • All languages supported
  • Inline chat + completions

Docker Compose

Bundles your whole app — database, backend, frontend — into one portable package. Run it on any computer with one command.

  • Free Docker Desktop
  • No setup differences
  • Deploy anywhere
  • Industry-standard skill
